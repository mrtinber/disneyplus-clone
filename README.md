## Installation

First fork the project:

```bash
git clone https://github.com/mrtinber/disneyplus-clone.git
cd disneyplus-clone
```

Then run npm install:

```bash
npm install
```

And then run live server:

```bash
npm run dev
```

You should be able to view the project at http://localhost:5173
